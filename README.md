# transportadora_backend
 
 Desenvolvido por Vanderson Teodoro Camatini e Nicolas Luz

 - Para iniciar o projeto, você deve ter o node instalado na sua máquina.
 - Rode o comando npm install ou yarn install, dependendo do seu gerenciador de pacotes.
 - Vá na pasta SQL, abra o documento database.sql, copie o código e cole no seu gerenciador do banco de dados para criar as tabelas
 - Após todas as dependencias do projeto criadas, e do banco de dados configurado. Rode o comando npm run dev ou yarn dev na aplicação backend e npm start ou yarn start na aplicação frontend.
